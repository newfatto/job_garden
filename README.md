# Job Garden

**Job Garden** — консольное Python-приложение для поиска вакансий через API hh.ru, сохранения данных в JSON-файл и работы с вакансиями через объектно-ориентированную архитектуру.

## Описание проекта

Проект разработан как курсовая работа по Python и демонстрирует интеграцию с внешним API, работу с JSON-файлами, построение архитектуры на основе ООП и покрытие функциональности тестами.

Приложение получает вакансии с hh.ru по пользовательскому запросу, преобразует данные API в объекты вакансий, сохраняет их в локальное JSON-хранилище и позволяет работать с результатами через консольный интерфейс: фильтровать вакансии, сортировать их по зарплате, искать по ключевым словам, выбирать top-N предложений и удалять ненужные записи.

Проект показывает навыки работы с абстрактными классами, наследованием, инкапсуляцией, API-клиентами, файловым хранилищем и тестированием бизнес-логики.

## Основной функционал

- Получение вакансий с hh.ru через внешний API.
- Поиск вакансий по ключевому слову.
- Преобразование данных API в объекты класса `Vacancy`.
- Нормализация данных вакансии:
  - название;
  - ссылка;
  - город;
  - работодатель;
  - зарплата;
  - валюта;
  - описание.
- Сравнение вакансий по уровню зарплаты.
- Сохранение вакансий в JSON-файл.
- Защита от дублирования вакансий при сохранении.
- Удаление вакансий из JSON-хранилища.
- Фильтрация вакансий:
  - по ключевым словам;
  - по городу;
  - по зарплате;
  - по валюте.
- Сортировка вакансий по зарплате.
- Вывод top-N вакансий.
- Консольное взаимодействие с пользователем.
- Покрытие функциональности тестами.

## Архитектура проекта

Проект построен по принципу разделения ответственности.

### API-слой

Модуль `src/api.py` отвечает за подключение к API hh.ru и получение данных о вакансиях.

API-клиент изолирован от остальной бизнес-логики: приложение может получать данные из внешнего источника, но дальнейшая обработка вакансий не зависит от формата ответа API напрямую.

### Абстрактные классы

Модуль `src/base.py` содержит абстрактные классы, которые задают общий интерфейс для:

- классов, работающих с API;
- классов, работающих с файловым хранилищем.

Это позволяет отделить контракт от реализации и делает структуру проекта более расширяемой.

### Модель вакансии

Модуль `src/vacancy.py` содержит класс `Vacancy`, который описывает объект вакансии.

Класс отвечает за:

- хранение данных вакансии;
- нормализацию входящих данных;
- валидацию зарплаты;
- преобразование объекта в словарь;
- сравнение вакансий по зарплате.

### JSON-хранилище

Модуль `src/files.py` содержит класс для работы с JSON-файлом.

Он отвечает за:

- чтение сохранённых вакансий;
- добавление новых вакансий;
- защиту от дублей;
- удаление вакансий;
- сохранение изменений в файл.

### Утилиты

Модуль `src/utils.py` содержит функции для обработки коллекции вакансий:

- фильтрация;
- сортировка;
- выбор top-N;
- обработка диапазона зарплат;
- преобразование данных.

## Установка и запуск

### 1. Клонировать репозиторий

```bash
git clone https://github.com/newfatto/job_garden.git
cd job_garden
```

### 2. Установить зависимости

Если используется Poetry:

```bash
poetry install
```

Активировать виртуальное окружение:

```bash
poetry env activate
```

Или запускать команды через Poetry:

```bash
poetry run python main.py
```

### 3. Настроить переменные окружения

Создайте файл `.env` на основе `.env_template`.

```bash
cp .env_template .env
```

Если проект использует дополнительные настройки или API-параметры, их можно указать в `.env`.

### 4. Запустить приложение

```bash
python main.py
```

Или через Poetry:

```bash
poetry run python main.py
```

## Пример пользовательского сценария

Пользователь запускает приложение и вводит параметры поиска:

```text
Введите поисковый запрос: Python
Введите город: Москва
Введите минимальную зарплату: 100000
Введите валюту: RUR
Введите количество вакансий для вывода: 10
```

Приложение:

1. отправляет запрос к API hh.ru;
2. получает список вакансий;
3. преобразует данные в объекты `Vacancy`;
4. сохраняет вакансии в JSON-файл;
5. фильтрует результаты по заданным параметрам;
6. сортирует вакансии по зарплате;
7. выводит top-N подходящих вакансий.
